Problem:              Solution:
─────────────────────────────────────────────────────────────────────────────
Site shows 404        → Check .htaccess exists (show hidden files)
Styles broken         → Clear cache, verify assets/ folder uploaded
Can't deploy          → Try different method (SSH instead of cPanel)
Services not working  → Verify environment variables set
Email not sending     → Check RESEND_API_KEY configured
WhatsApp not working  → Setup Twilio webhook (see guide)
Other issues          → Check browser console (F12 → Console)
